# Project Map

This template expects configurable project files. Common defaults:

```text
PROJECT_STATE.md or CURRENT_STATE.md
WORK_QUEUE.md
PROJECT_INDEX.md or FILES.md
workflow/
drafts/
readable/ or de-ai/
summaries/
audits/
ledgers/
outlines/
timelines/
maps/
lore/
external-data/
standards/
context-packs/
branches/
reports/
exports/
bible/ or canon/
characters/
skill/ or skills/
```

## Canon assets

- Draft chapters.
- Readable/publishable chapters.
- Summaries.
- Formal audits.
- Ledgers and continuity locks.
- Outlines and canon bible.
- Timeline, geography, lore, branch-simulation, and generated export files when intentionally included.
- Active workflow files.

## Process assets

- scratch/model outputs;
- inbox/outbox transfer files;
- raw logs;
- old archives.

These are not default context and should not be packaged for public release unless explicitly sanitized.

Generated Org outlines default to `exports/org/project-outline.org`. They are navigation aids, not canon approval.
