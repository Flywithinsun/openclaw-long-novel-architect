# Portability Guide

## Source project

1. Create or update `novel-architect.config.json`.
2. Verify assets.
3. Package assets.
4. Inspect manifest.
5. Run a secret scan.

## Destination project

1. Unpack the asset package.
2. Install/copy the Skill.
3. Verify assets.
4. Map model roles to available routes.
5. Confirm current stop point from files.
6. Only then start chapter work.

## Skill alone is not enough

The Skill describes the method. The project assets provide facts: current chapter, canon, outlines, and continuity.
