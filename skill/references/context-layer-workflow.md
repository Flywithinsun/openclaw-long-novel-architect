# Context Layer Workflow / 上下文层工作流

Use this workflow to keep historical prose grounded in explicit standards, canon files, timelines, lore, and current manuscript state.

## Context layers

Layer 1: Standards

- `standards/prose-style.md`
- `standards/historical-dialogue.md`
- `standards/forbidden-modernisms.md`
- `standards/deai-style-rules.md`
- `standards/chapter-rhythm.md`

Layer 2: Canon / research

- `canon/`
- `outlines/`
- `characters/`
- `timelines/`
- `lore/`

Layer 3: Manuscript state

- `drafts/`
- `readable/`
- `summaries/`
- `audits/`
- `ledgers/`
- `PROJECT_STATE.md`
- `WORK_QUEUE.md`

## Before drafting

1. Read configured standards first.
2. Read relevant timelines and lore cards.
3. Build or refresh a chapter context pack.
4. Put period-language constraints, forbidden modernisms, canon locks, and open risks into the chapter request.

## Before de-AI / readability work

Read `standards/deai-style-rules.md` if present. Do not smooth away period voice, social hierarchy, ritual language, or character-specific diction.

## Stop rule

Passing a context check does not authorize the next chapter. Report changed files and stop.
