# PROJECT_INDEX

## Required files

- `PROJECT_STATE.md` — current project status and stop point.
- `WORK_QUEUE.md` — active and upcoming tasks.
- `PROJECT_INDEX.md` — this file map.
- `workflow/model-routing.md` — model role mapping.
- `skill/SKILL.md` — OpenClaw operating rules.

## Directories

- `drafts/` — canonical draft chapters.
- `readable/` — readable / de-AI polished candidates.
- `summaries/` — chapter or range summaries.
- `audits/` — range audits and quality reviews.
- `ledgers/` — continuity, economy, logistics, institutions, promises, or other ledgers.
- `outlines/` — arc and chapter outlines.
- `characters/` — character files.
- `canon/` — canon bible and hard rules.
- `writing-requests/` — self-contained chapter request packages.
- `exports/` — intentional export outputs.

## Naming conventions

- Draft chapter: `drafts/chNNN.md`
- Readable chapter: `readable/chNNN.md`
- Summary: `summaries/chNNN-summary.md`
- Audit: `audits/chAAA-chBBB-audit.md`

Adjust these conventions in `novel-architect.config.json` if your project uses different paths.
