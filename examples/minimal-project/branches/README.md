# Branches / 分支模拟

This directory holds alternate-history or experimental branch simulations.

这个目录用于存放架空历史或实验性分支模拟。

## Workflow

1. Write a divergence point.
2. Run logic audit if the branch changes institutions, war, economy, logistics, geography, or social order.
3. Write a simulated outline.
4. Write consequences.
5. Write a merge decision.
6. Update `WORK_QUEUE.md` and `PROJECT_STATE.md` if the branch is accepted.

## Example branch

```text
branches/branch-history-A/
```

## Stop rule

Branch simulation is not canon by default. Wait for a merge decision before writing canon prose.
