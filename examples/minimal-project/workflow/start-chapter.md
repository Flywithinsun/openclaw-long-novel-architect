# Start Chapter Workflow

Before drafting a chapter, OpenClaw should:

1. Confirm the target chapter and scope.
2. Read `PROJECT_STATE.md`, `WORK_QUEUE.md`, and `PROJECT_INDEX.md`.
3. Read `workflow/model-routing.md`.
4. Read relevant canon, outline, character, and ledger files.
5. Produce a self-contained request package in `writing-requests/`.
6. Wait for approval if the user requested request-only mode.
